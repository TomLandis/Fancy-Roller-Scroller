This simple plugin alows you to easly manage and implement a rolling text list.

== Description ==

The plugin lets you have a title above your list with words or phrases below rotating in a scrolling rotation.
== Installation ==

This section describes how to install the plugin and get it working.
1) Install the plugin on your site through the 'add plugin' menu.
2) Click 'Activate'.
3) go to 'Settings' a new entry should appear for 'Fancy Roller Scroller'.
4) On this page you can enter the contents of your list and push 'Save'.
5) The shortcode [fancy_roller_scroller] can be placed on any page or post to imbed your new scroller.
6) CSS classes are provided with the list items '.frs-list-item' and the top text 'frs-top-text' use these to target your styles!
